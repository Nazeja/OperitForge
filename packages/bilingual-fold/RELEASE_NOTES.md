# 双语折叠 v0.1.0

首次发布。

## 功能

- 支持 `<fold>...</fold>` 标签折叠渲染译文。
- 原文保持 Operit 原生气泡样式，插件只渲染 `<fold>` 内文本。
- 折叠状态显示可点击按钮，展开后按钮原地替换为译文。
- 点击译文本身可再次收起。
- 支持输入栏菜单临时开启 / 关闭折叠渲染。
- 支持配置默认展开译文。
- 支持配置折叠按钮点击区域宽度。
- 支持自定义折叠按钮样式。
- 支持内置双语提示词复制。
- 配置使用 Operit ToolPkg env 持久化。
