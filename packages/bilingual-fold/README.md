# 双语折叠（Bilingual Fold）

双语模式插件，在对话中支持 `<fold>译文</fold>` 标签，将译文渲染为可展开/收起的按钮，并提供自定义选项。

## 当前版本

`0.1.0`

## 主要功能

- 支持在聊天内容中使用 `<fold>译文</fold>` 标签折叠译文。
- 原文保持 Operit 原生气泡样式，插件只渲染 `<fold>` 内的译文。
- 折叠状态显示可点击的小按钮；展开后按钮原地替换为译文。
- 点击译文本身可再次收起。
- 支持输入栏菜单临时开启 / 关闭折叠渲染。
- 支持配置新消息是否默认展开译文。
- 支持配置折叠按钮点击区域宽度：40% / 50% / 60% / 70% / 80%。
- 支持自定义折叠按钮文字，并提供几组预设样式。
- 内置一份双语模式提示词，可复制到角色卡、世界书、用户偏好或系统提示中。
- 不自动注入提示词，保持低侵入。

## 标签格式

将原文正常写在消息里，把译文放入 `<fold>...</fold>`：

```text
The rain had stopped, but the city still smelled like thunder.
<fold>雨已经停了，但整座城市仍然闻起来像雷声。</fold>
```

多行内容也可以放在同一个 `<fold>` 中：

```text
He did not answer immediately.
For a moment, only the sea spoke between them.
<fold>他没有立刻回答。
有那么一瞬间，只有海在他们之间说话。</fold>
```

插件本身不限制语言方向。你可以用于外文→中文、中文→英文，或其他双语场景。

## 配置项

在 Operit 的插件管理 / 工具箱入口打开“双语折叠设置”。

### 默认展开译文

控制收到新消息时 `<fold>` 内容是否默认展开。

- 关闭：默认显示折叠按钮，需要点击展开。
- 开启：默认直接显示译文，需要点击译文收起。

### 折叠按钮点击区域宽度

控制折叠按钮左侧可点击区域宽度。

可选：

```text
40% / 50% / 60% / 70% / 80%
```

右侧会保留区域用于长按消息菜单。

### 折叠按钮样式

可以自定义折叠状态下显示的小按钮文字。

- 最多 16 个字符。
- 留空会恢复默认值：`▸▹▸▹`。
- 配置页内置若干预设，可直接点击套用。

## 内置双语提示词

插件不会自动注入提示词。

如需让模型自动按双语格式输出，可在配置页复制内置提示词，并自行放入：

- 角色卡
- 世界书
- 用户偏好
- system prompt
- 其他你希望控制输出格式的位置

## 已知限制

- 输入栏“双语折叠”开关是临时运行状态。清后台或重启插件后会恢复默认开启。
- 插件只负责渲染 `<fold>` 标签，不会自动让模型输出双语内容。
- 如果模型把 `<fold>` 标签放进 Markdown 代码块，Operit 可能不会按 XML 标签渲染。
